# Basic Calculator (1.0.0)

**Basic Calculator** is a simple terminal-based program that performs simple arithmetic operations.

## Usage

1. Execute the program.
2. The menu will display with operation options (1-4).
3. Enter your selection (1 for Addition, 2 for Subtraction, 3 for Multiplication, 4 for Division).
4. Enter the first number when prompted.
5. Enter the second number when prompted.
6. The result will be displayed and the program will exit.

## Features

- **Addition**: Add 'a' and 'b' together.
- **Subtraction**: Subtract 'a' from 'b'.
- **Multiplication**: Multiply 'a' and 'b' together.
- **Division**: Divide 'a' from 'b'.
- **Input Validation**: Properly handles invalid inputs. (hopefully)
