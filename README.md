## Basic Calculator (1.0.0)

**Basic Calculator** is a terminal-based program designed to perform simple arithmetic operations. Users can easily select an operation (addition, subtraction, multiplication, or division) and input two numbers to receive the calculated result displayed with two decimal places.

### Features

- **Addition**: Add two numbers together.
- **Subtraction**: Subtract one number from another.
- **Multiplication**: Multiply two numbers.
- **Division**: Divide one number by another.
- **Input Validation**: Ensures valid menu selection and numeric input.
- **Formatted Output**: Results are displayed with two decimal places.

### Usage

1. Execute the program.

2. The menu will display with operation options (1-4).

3. Enter your selection (1 for Addition, 2 for Subtraction, 3 for Multiplication, 4 for Division).

4. Enter the first number when prompted.

5. Enter the second number when prompted.

6. The result will be displayed and the program will exit.
